ZOODOM uses the **Onest** font family from Google Fonts, providing a modern and clean typographic system.

## Font Family

### Onest Font

The Onest font is a variable font supporting weights from 100 to 900, loaded from Google Fonts.

<Info>
Onest is applied globally to all HTML elements through the `html, body` selector.
</Info>

### Font Import

The font is imported in `Head.astro` using Google Fonts:

```html
<!-- Preconnect to Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<!-- Onest Font -->
<link href="https://fonts.googleapis.com/css2?family=Onest:wght@100..900&display=swap" rel="stylesheet">
```

<Note>
The `wght@100..900` parameter loads all available font weights (variable font).
</Note>

### Global Font Declaration

In `global.css`, Onest is applied to all elements:

```css
html, body {
    font-family: 'Onest', sans-serif;
    background-color: var(--color-neutral-white-900);
    color: var(--color-neutral-black-900);
}
```

## Heading Styles

### H1 Styling

The main heading (H1) uses large, bold typography for maximum impact:

```css
h1 {
    font-size: 64px;
    font-weight: 700;
    color: var(--color-neutral-black-900);
}
```

## Paragraph Styles

Paragraph text uses comfortable reading size with spacing:

```css
p {
    font-size: 18px;
    color: var(--color-neutral-gray-900);
    margin-top: 20px;
    margin-bottom: 30px;
}
```

## Font Weights Available

Since Onest is loaded as a variable font (100-900), you can use any weight:

```css
/* Thin */
.text-thin { font-weight: 100; }

/* Light */
.text-light { font-weight: 300; }

/* Regular */
.text-regular { font-weight: 400; }

/* Medium */
.text-medium { font-weight: 500; }

/* Semi-Bold */
.text-semibold { font-weight: 600; }

/* Bold */
.text-bold { font-weight: 700; }

/* Extra Bold */
.text-extrabold { font-weight: 800; }

/* Black */
.text-black { font-weight: 900; }
```

## Special Typography Styles

### Active Navigation Links

Active navigation links receive special styling:

```css
a.active {
    color: var(--color-brand-green-800) !important;
    font-weight: 600;
}
```

<Info>
Active links use semi-bold weight (600) and the primary brand green color.
</Info>

### Usage in Navigation

```html
<a href="/" class="active">Home</a>
```

## Complete Typography System

```css
/* Font Family */
html, body {
    font-family: 'Onest', sans-serif;
    background-color: var(--color-neutral-white-900);
    color: var(--color-neutral-black-900);
}

/* Heading 1 */
h1 {
    font-size: 64px;
    font-weight: 700;
    color: var(--color-neutral-black-900);
}

/* Paragraphs */
p {
    font-size: 18px;
    color: var(--color-neutral-gray-900);
    margin-top: 20px;
    margin-bottom: 30px;
}

/* Active Links */
a.active {
    color: var(--color-brand-green-800) !important;
    font-weight: 600;
}
```
