ZOODOM uses a comprehensive color system built with CSS custom properties (variables) defined in `global.css`. The palette includes brand greens, accent greens, extended colors, and neutral tones.

## CSS Variables Overview

All colors are defined as CSS variables in the `:root` selector, making them globally accessible throughout the application.

```css
:root {
    /* CSS variables defined here */
}
```

To use any color in your styles:

```css
.my-element {
    background-color: var(--color-brand-green-800);
    color: var(--color-neutral-white-900);
}
```

```css

/* Black overlay */

```

<Note>
The overlay color (`#00000077`) has 77 transparency for semi-transparent backgrounds.
</Note>

## Usage Examples

### In Custom CSS

```css
.custom-button {
    background-color: var(--color-brand-green-800);
    color: var(--color-neutral-white-900);
}

.custom-button:hover {
    background-color: var(--color-brand-green-700);
}
```

### In Astro Components

```html
<div style={`background-color: var(--color-brand-green-50); color: var(--color-neutral-black-900);`}>
    Content with brand colors
</div>
```

### Text Color Utility

The `.text-primary` class applies the brand green color:

```css
.text-primary {
    color: var(--color-brand-green-800);
}
```

```html
<p class="text-primary">This text is brand green</p>
```
