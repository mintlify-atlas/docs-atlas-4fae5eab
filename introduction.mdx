# Introduction

ZOODOM is a modern zoo website for the **Parque Zoológico Nacional** (National Zoo Park), built with [Astro](https://astro.build) and Bootstrap 5. The project showcases a clean, responsive design focused on providing an excellent user experience for visitors looking to explore the zoo's offerings.

## What is ZOODOM?

ZOODOM is a static website built to help visitors discover wildlife and plan family trips to the national zoo park. The site features a beautiful hero section, responsive navigation, and is optimized for Spanish-speaking audiences in the Dominican Republic.

## Technology Stack

ZOODOM is built with modern web technologies:

- **Astro 6.0.4** - Fast, content-focused framework
- **Bootstrap 5.3.8** - Responsive CSS framework
- **Node.js 22.12.0+** - Runtime environment
- **Onest Font** - Custom Google Fonts integration

<Info>
  The site is configured to be deployed at `https://zoodomi.netlify.app` with full static site generation.
</Info>

## Project Structure

The project follows Astro's standard structure:

```plaintext
zoodom/
├── src/
│   ├── pages/         # Route pages (index.astro)
│   ├── layouts/       # Layout components (Layout.astro)
│   ├── components/    # Reusable components (Head.astro)
│   └── assets/        # Images and styles
├── public/            # Static assets
├── astro.config.mjs   # Astro configuration
└── package.json       # Dependencies
```
